# Content Model Guide

This guide covers the basics for creating and managing Contentful content types and fields.

## Content types

A content type defines a structure for entries. Create a new content type when you need a new object type, such as blog posts, authors, or products.

### Recommended steps

1. Open Contentful and navigate to the Content Model section.
2. Click `+ Add content Type`.
3. Give the content type a name and description.
4. Choose a display field, usually `title`, `name`, or another primary identifier.

## Fields
### Rich text
- Rich text editing. You can have editing options in it and embed entries or assets directly into the field.

### Text (Symbol)
- Plain text. You can select between Long and Short text options.
- Text Field Type
    1. **Short Text**
    2. **Long Text**
    3. **List**

### Number
- Number values, e.g. price, ratings, length, pixels, spacing etc.
- Field Type
    1. **Integer**
    2. **Decimal**

### Date and time
- Date, time, and timezone data.

### Location
- Information on geo-coordinates or location.

### Media
- Adding images to your entry field through assets.
    1. One File
    2. Many File

### Boolean
- Enabling an editor to select one of the two options. For example, this field can be used for NoFollow or NoIndex tags.

### JSON
- JSON object. This field is helpful for apps.

### Reference
- Link entries or assets to your entry
    1. One Reference
    2. Many Reference
